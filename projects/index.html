InsightLoop

InsightLoop ist eine KI-gestützte Webanwendung zur Analyse von Kundenkommunikation im E-Commerce.

Die Anwendung klassifiziert Support-Nachrichten automatisch und generiert umsetzbare Insights zur Verbesserung von Geschäftsprozessen.

Ruby on Rails · PostgreSQL · OpenAI API · RAG · HTML · CSS
